Приложения клиента и сервера работают с базой данных коллекции. Для того чтобы начать работу, введите порт на обоих приложениях. 

Памятка по клиентскому приложению:
Чтобы получить списоок комманд, введите команду help. Клиент должен авторизоваться, прежде чем работать с коллекцией. Ввод команд происходит с помощью агрумента командной строки.  Имя файла должно передаваться программе также с помощью аргумента командной строки, как, например, в команде execute_script.

Памятка по серверному приложению:
Команды на сервере:
save - сохраняет коллекцию в БД
exit - закрывает приложение, предварительно сохраняя коллекцию в БД

P.S. Учтите! При экстренном выходе (ctrl+D, выход из консоли и т.д.) коллекция не будет сохранена!
